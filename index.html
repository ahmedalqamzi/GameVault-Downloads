Good to know

Questions before you install.

GameVault is a small independent app shared directly, not a store release or a public social network.

Does it work without any online service?

Yes. Android, iPhone, and the installed web app keep an on-device library without an account, token, PC, or personal server. Search and metadata need internet. Private cloud backup is optional; Steam library import additionally needs its server-side Steam API key.

Can I import my previous spreadsheet?

Yes. Export the Gamery library as CSV, then import it from Settings in Android or the web app. GameVault preserves mapped folders, ratings, reviews, progress, dates, summaries, and platforms.

How does Steam sign-in work?

GameVault opens Steam's own sign-in page through OpenID and receives only your Steam ID—never your password. When the service has its required Steam Web API key, it can import the owned games and playtime allowed by your Steam Game Details privacy setting. If no key is configured, GameVault hides that connection flow instead of offering a setup that cannot finish. Steam tags, controller support, and ProtonDB compatibility still work from a saved game's Steam store link.

Will an APK update erase my library?

No. Install the newer signed APK over the existing app and Android retains its data. Do not uninstall first. For an extra copy or a new phone, use Settings → Export full backup, then restore that JSON file.

Are trailers linked for every game?

GameVault shows linked YouTube trailers whenever IGDB metadata provides them. Some older, unreleased, or obscure games may not have a usable trailer.

What is included in franchise timelines?

They are designed around obtainable base games. DLC, bundles, cancelled entries, and broadcast-only titles are excluded. Remasters are combined with the original; remakes stay separate.

Will cloud sync combine my friends' libraries?

No. The cloud setup is deliberately single-owner. Each person should deploy their own database and keep their token private.

Why does iPhone need SideStore or AltStore?

The IPA is shared outside the App Store and is unsigned. A sideloading app signs it for your device. Free Apple signing generally needs refreshing every seven days.
